This calculator contains 9 fields, 4 trigonometric buttons, 4 classic arithmetic buttons, 3 exponential buttons and 3 control buttons.
---===#===---
			[CLEAR]
[F1]	[F2]	[F3]	[SAVE]
[F4]	[F5]	[F6]	[LOAD]
[F7]	[F8]	[F9]	

[COS][CTG]    [pow2]	[-][/]
[SIN][TAN] [pow3][powN]	[+][*]
---===#===---
F1-F9 are changeable and numbers have to look like "X,X", where X is chain of numbers 0-9.
---===#===---
CLEAR deletes everything from F1-F9.
---===#===---
SAVE saves values of F1-F9 into .calculator file (it will be near .sln).
---===#===---
LOAD uses .calculator to return values of F1-F9 to saved state (if there is no file, it does nothing).
---===#===---
COS, SIN, CTG, TAN apply trigonometric transformations to all F1-F9.
---===#===---
pow2, pow3 also apply to all F1-F9.
---===#===---
powN has another logic:
x^y=z, where x - F1,F4,F7; y - F2,F5,F8; z - F3,F6,F9.
---===#===---
+ has similar logic: x+y=z
-: x-y=z
/: x/y=z
*: x*y=z
where x - F1,F4,F7; y - F2,F5,F8; z - F3,F6,F9.
---===#===---
